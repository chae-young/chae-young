1) v-show 조건에 맞지 않으면 style에 display:none을 추가해서 안보이게 만든다. 2) v-if 조건에 맞지 않으면 렌더링하지 않는다. 따라서 우리는 앞으로 조건에 따라 아예 코드를 지우고 싶다면 v-if 디렉티브를, 코드는 있지만 스타일로 display: none 처리를 하고 싶다면 v-show를 사용하여야 할 것입니다. v-if 디렉티브는 조건상태가 true 이면 해당태그를 렌더링하고, false인 경우 렌더링을 하지 않는 디렉티브이 v-if 는 조건부 블럭 안의 내용에 따라 제거되고 만들어지고 v-show 는 우선 렌더링이 되고 CSS 속성으로 감추고 보여줌 따라서 v-if 는 토글 할 때의 비용이 높고, v-show 는 초기 렌더링 비용이 높으므로 자주 바뀐다면 v-show , 바뀌지 않는다면 v-if 를 권장함 v-if와 v-for 양방향 데이터 바인딩 v-model (폼에 관련된 태그 에 사용) Vue 인스턴스 ⇄ Template
